# Язык 67 (интерпретатор на Python)

`67` — маленький язык программирования с русским синтаксисом: переменные, арифметика, `если/иначе`, цикл `пока`.

## Запуск

```bash
python core/lang67.py example.67
```

## Синтаксис (только русский)

- Объявление/изменение переменной:
  - `пусть x = 10`
- Вывод:
  - `вывод x`
- Ввод:
  - `ввод x` (введи число или текст в консоли)
- Арифметика:
  - `+`, `-`, `*`, `/` (целочисленное деление)
- Строки:
  - `пусть имя = "Иван"`
  - `вывод "Привет, " + имя`
- Условия:
  - `==`, `!=`, `<`, `>`, `<=`, `>=`
  - логика: `и`, `или`, `не`
- Условный блок:
  - `если условие то ... иначе ... конец`
- Цикл:
  - `пока условие делать ... конец`
- Управление циклом:
  - `стоп` (выйти из цикла)
  - `продолжить` (перейти к следующей итерации)
- Комментарии:
  - `# это комментарий`

## Пример программы

```txt
пусть число = 3
пусть результат = 1
пока число > 0 делать
    пусть результат = результат * число
    пусть число = число - 1
конец
вывод результат
```
