# Язык 67 (интерпретатор на Python)

`67` — учебный мини-язык программирования с русским синтаксисом.  
В проекте реализован полноценный цикл: лексер -> парсер -> интерпретатор.

Поддерживаются:
- переменные и выражения;
- числа и строки;
- условия `если/иначе`;
- циклы `пока`;
- команды `стоп` и `продолжить`;
- ввод/вывод через консоль.

## Требования

- Python 3.10+ (рекомендуется 3.11/3.12)
- Windows / Linux / macOS

Внешние зависимости не нужны.

## Быстрый старт

1. Перейди в папку проекта.
2. Запусти интерпретатор с файлом `.67`:

```bash
python core/lang67.py simple.67
```

Если хочешь использовать свой файл:

```bash
python core/lang67.py my_program.67
```

## Структура проекта

- `lang67.py` — весь интерпретатор (токенизация, парсинг, выполнение).
- `simple.67` — демонстрационная программа.
- `How to start.txt` — краткая команда запуска.

## Синтаксис языка 67

Все ключевые слова языка — на русском.

### Переменные

Объявление и переопределение:

```txt
пусть x = 10
пусть x = x + 5
```

### Вывод

```txt
вывод x
вывод "Привет"
```

### Ввод

```txt
ввод имя
вывод имя
```

`ввод` пытается преобразовать значение к числу (`int`). Если не получилось — сохраняет как строку.

### Типы данных

- число (целое): `1`, `42`, `-7`
- строка: `"текст"`

### Арифметика

- `+`, `-`, `*`, `/`
- деление `/` — **целочисленное** (`//`)

Пример:

```txt
пусть a = 7
пусть b = 2
вывод a / b   # выведет 3
```

### Строки

```txt
пусть имя = "Иван"
пусть привет = "Привет, " + имя
вывод привет
```

Важно: сложение разрешено только для `число+число` или `строка+строка`.

### Условия

Операторы сравнения:
- `==`, `!=`, `<`, `>`, `<=`, `>=`

Логические операторы:
- `и`, `или`, `не`

```txt
если не (x == 0) и (x > 3 или x == 1) то
    вывод "условие истинно"
иначе
    вывод "условие ложно"
конец
```

### Ветвление

```txt
если x > 0 то
    вывод "положительное"
иначе
    вывод "не положительное"
конец
```

### Цикл

```txt
пусть x = 0
пока x < 3 делать
    вывод x
    пусть x = x + 1
конец
```

### Управление циклом

- `стоп` — выход из цикла;
- `продолжить` — переход к следующей итерации.

```txt
пусть x = 0
пока x < 5 делать
    пусть x = x + 1
    если x == 2 то
        продолжить
    иначе
        вывод x
    конец
    если x == 4 то
        стоп
    иначе
        вывод 0
    конец
конец
```

### Комментарии

Комментарий начинается с `#` и действует до конца строки:

```txt
# это комментарий
пусть x = 1  # и это тоже комментарий
```

## Ограничения текущей версии

- только целые числа (`int`);
- нет функций и пользовательских процедур;
- нет списков/словарей/модулей;
- нет `для`-цикла, только `пока`;
- нет escape-последовательностей в строках.

## Типичные ошибки

- `Неизвестная переменная 'x'` — переменная не была объявлена через `пусть`.
- `Деление на ноль` — в выражении есть деление на 0.
- `Команда 'стоп' ... только внутри цикла` — `стоп` использован вне `пока`.
- `Команда 'продолжить' ... только внутри цикла` — `продолжить` использован вне `пока`.
- `Неизвестный символ ...` — в коде есть неподдерживаемый символ.

## Полный пример

Файл `simple.67`:

```txt
# Демонстрация новых команд языка 67
пусть имя = "Мир"
вывод "Привет, " + имя

пусть x = 0
пока x < 6 делать
    пусть x = x + 1
    если x == 2 то
        продолжить
    иначе
        вывод x
    конец
    если x == 4 то
        стоп
    иначе
        вывод 0
    конец
конец

если не (1 == 2) и (3 > 2 или 0 == 1) то
    вывод "Логика работает"
иначе
    вывод "Ошибка логики"
конец
```
